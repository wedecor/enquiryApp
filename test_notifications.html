Notification Debugging

Open Chrome DevTools Console (F12) and run these commands:

1. Check if you have FCM token:

// In Chrome Console, run:
firebase.messaging().getToken().then(token => {
    console.log('FCM Token:', token);
    if (!token) {
        console.error('❌ NO FCM TOKEN - Request permission first!');
    } else {
        console.log('✅ FCM Token found:', token);
    }
});

2. Request notification permission:

// In Chrome Console, run:
Notification.requestPermission().then(permission => {
    console.log('Notification permission:', permission);
    if (permission === 'granted') {
        console.log('✅ Permission granted!');
    } else {
        console.error('❌ Permission denied:', permission);
    }
});

3. Check Firestore for your notifications:

Go to Firebase Console → Firestore → users/{yourUserId}/notifications

You should see notification documents if they're being created.

4. Check Firestore for your FCM tokens:

Go to Firebase Console → Firestore → users/{yourUserId}/private/notifications/tokens

You should see at least one token document.
